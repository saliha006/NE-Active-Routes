# NE-Active-Routes
A HTML/CSS based website that encourages users in the north east to be more active.

# *This mobile webpage features:*

- search bar with user points and level 
- hero section 
- infrastructure and impacted routes 
- discover routes section
- events and challenegs sections
- bike help repairline 
- floating create routes button
- bottom navigation bar

## *technologies used:*

- HTML
- CSS

### *How to run the webpage*
- live preview or live server to run
- right click mobile.html
- then open with lve server
